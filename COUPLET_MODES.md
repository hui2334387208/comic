# 对联生成模式完整指南

## 支持的生成模式（27种）

### 基础模式（7种）

### 1. 基础主题模式
**输入格式**：简单主题词
**示例**：
- `春节` → 生成春节主题对联
- `中秋` → 生成中秋主题对联
- `生日` → 生成生日主题对联
- `乔迁` → 生成搬家主题对联

### 2. 具体场景模式
**输入格式**：详细场景描述
**示例**：
- `餐厅开业典礼` → 生成餐厅开业专用对联
- `婚礼现场布置` → 生成婚礼现场对联
- `公司年会庆典` → 生成企业年会对联
- `学校校庆活动` → 生成校庆专用对联

### 3. 上联对下联模式
**输入格式**：直接输入上联，或标注"上联："
**示例**：
- `春风得意马蹄疾` → 对出下联"一日看尽长安花"
- `上联：千门万户曈曈日` → 对出下联"总把新桃换旧符"

### 4. 下联对上联模式
**输入格式**：直接输入下联，或标注"下联："
**示例**：
- `一日看尽长安花` → 对出上联"春风得意马蹄疾"
- `下联：总把新桃换旧符` → 对出上联"千门万户曈曈日"

### 5. 横批生成模式
**输入格式**：输入完整上下联
**示例**：
- `千门万户曈曈日，总把新桃换旧符` → 生成横批"辞旧迎新"

### 6. 藏头对联模式
**输入格式**：`藏头：[指定字词]`
**示例**：
- `藏头：张三` → 生成"张灯结彩迎新春，三阳开泰庆丰年"
- `藏头：恭喜发财` → 生成四句藏头对联

### 7. 嵌字对联模式
**输入格式**：`嵌字：[指定字词]`
**示例**：
- `嵌字：恭喜发财` → 在对联中间位置嵌入这些字
- `嵌字：福禄寿喜` → 生成包含这些吉祥字的对联

## 对联格律要求

每副对联必须符合传统格律要求：
- **字数相等，断句一致** - 上下联字数必须相同
- **词性相对，位置相同** - 名词对名词，动词对动词等
- **平仄协调，上仄下平** - 上联末尾字为仄声，下联末尾字为平声
- **意思相关，内容呼应** - 上下联内容要有关联

## 使用技巧

1. **组合使用**：可以组合多种模式，如"北京餐厅开业 古典风格 七字联"
2. **明确标注**：使用明确的标注词可以提高识别准确率
3. **具体描述**：越具体的描述，生成的对联越贴合需求
4. **文化背景**：可以加入特定的文化背景要求
